# EWArcTableView-OC
[![996.icu](https://img.shields.io/badge/link-996.icu-red.svg)](https://996.icu)

OC版左侧有弧形区域的TableView

# 实现效果:

左侧有弧形区域的tableview,实现简单屏幕适配,可修改弧度和cell高度.

可增加中间cell最大,上下端cell依次变小的效果.
